## 프로젝트 목적
공부한 스프링 프레임워크의 원리를 보다 더 깊이 이해하고, 복습하기 위한 간단한 프로젝트입니다.
### 주 목적
1. 객체 지향적인 프로젝트 설계
2. Exception Handling, Validation, AOP 등 스프링 기본 원리 및 기술
3. JPA 적용 및 쿼리 성능 확인
4. 테스트 작성 연습
5. OpenAPI Spec 적용, 문서화

## 프로젝트 스펙
- Java 17
- Springboot 3.2.5
- H2 database
- JPA
